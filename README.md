# dot-zsh

My zsh configuration

## Install
This zsh config is available as [Ellipsis][Ellipsis] package.

#### Ellipsis
``` shell
    ellipsis install sskytyba/dot-zsh
```

## Maintain
To keep everything up-to-date;

#### Ellipsis
```shell
    ellipsis pull zsh
```
